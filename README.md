<h4 align="center">Repository for storing contracts for Seed Crowdfunding.</h4>

### Install

```
git clone https://github.com/KnowledgeGRIDS/Seed.git
cd Seed
npm i
```

### Tests

```
npm run test
```

### Build contracts

```
npm run build:contracts
```

### Contracts' Tests

```
npm run test:contracts
```
